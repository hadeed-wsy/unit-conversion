Unit Converter Web App

A simple web-based unit converter built using Flask.

Features
Convert between multiple units:
Length: mm, cm, m, km, inch, foot, yard, mile
Weight: mg, g, kg, ounce, pound
Temperature: Celsius, Fahrenheit, Kelvin
Dynamic dropdowns (units update based on category)
Handles invalid inputs and unsupported conversions
Lightweight (no external libraries beyond Flask)
Tech Stack
Backend: Flask
Frontend: HTML + JavaScript
Language: Python
Project Structure
unit-conversion/
├── app.py
└── templates/
    └── index.html

Installation & Setup
1. Clone / Navigate
cd unit-conversion
2. Install dependencies
pip install flask
3. Run the app
python app.py
4. Open in browser
http://127.0.0.1:5000
